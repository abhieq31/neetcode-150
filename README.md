# NeetCode 150 — the honest count

**Solved so far: 2 of 150.**

That number is the whole point of this repo. Intellectual honesty means being
honest with yourself about what you don't know — and right now, the gap is
148 problems wide. Most solution repos exist to look finished. This one exists
to *get* finished, in public, where the count can't lie.

## The rules

- **One problem at a time, done properly.** It's not about how many things you
  start; it's about what you finish. A solution lands here only when it's
  accepted — no half-attempts, no copied answers.
- **The reps are the product.** You don't know you have grit until it's tested.
  This is the test, 150 questions long.
- **Auto-synced, not curated.** Solutions flow in automatically from
  [NeetCode.io](https://neetcode.io)'s GitHub Sync the moment they're accepted —
  so the repo is the raw signal of the practice, not a portfolio arranged
  after the fact.
